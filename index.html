Connect to Google Sheets

Follow these steps to connect your live Country Farms spreadsheet to this dashboard. Once connected, click "Refresh data" any time to pull the latest numbers.

1
Import the spreadsheet into Google Sheets

Go to sheets.google.com → File → Import → Upload → select CountryFarms_Ledger_2026.xlsx or CountryFarms_Ledger_2027.xlsx → choose "Insert new sheets".

2
Publish each sheet to the web as CSV

In Google Sheets → File → Share → Publish to web. For each sheet (Jan-Log, Jan-Summary, etc.) select the sheet name, choose CSV format, click Publish. Copy each URL.

3
Paste your Google Sheets URL below (or just the Sheet ID)

The dashboard extracts the Sheet ID automatically. Just paste the full URL from your browser — e.g.
https://docs.google.com/spreadsheets/d/YOUR_SHEET_ID/edit
Note: The 2026 sheet ID is already saved. This step is only needed for 2027.

4
Replace customer names in the spreadsheet

In each Jan-Log → Dec-Log sheet, update Row 3 columns D–K with your actual customer names. Use Find & Replace (Ctrl+H) to do all sheets at once.

Your URL is saved only in your browser's local storage and never sent anywhere. The dashboard reads data directly from Google's servers when you refresh.
